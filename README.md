# Agente-Inteligente-OCC-
Desarrollo academico de un agente inteligente con google adk capaz de generar informes académicos sobre el Modelo OCC con estructura formal y coherencia interna. El sistema integra un flujo automatizado que recupera contenido, redacta secciones extensas, contabiliza palabras y construye un documento final. Además, incorpora validaciones técnicas para asegurar requisitos como número mínimo de palabras, secciones obligatorias y referencias bibliográficas. Finalmente, el agente produce un JSON estructurado que representa el informe y permite su evaluación automática mediante criterios definidos.

## Características
- El sistema se caracteriza por automatizar la generación de informes académicos con estructura formal, integrando herramientas para recuperar contenido, redactar secciones y validar requisitos como extensión mínima y coherencia. Además, produce simultáneamente un PDF final y un JSON estructurado, facilitando tanto la presentación del documento como su evaluación automática.

## Tecnologías utilizadas
- Python, JSON, VSCode (IDE)
- Liberías: Google adk, csv, json, LiteLlm, dataclasses
